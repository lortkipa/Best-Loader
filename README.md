# About the project
BestLoader is UEFI bootloader, wich is part of BestOS, but it can be used for costum kernels too.  
It's written from scratch, using x86_64 assembly for performance reasons and to use [Best ABI](https://github.com/lortkipa/Best-ABI), which wouldn't be possible using existing compilers. 

# Platform requirments
* UEFI support
* x86_64 CPU
